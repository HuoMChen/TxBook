移植外部模块库到工程¶

外部模块库¶

外部模块包库含5个文件夹

  • demo 这里包含一些常用接口的用列。

  • dirver 这个文件夹包含用户需要实现的接口文件，根据每个平台不一样，用户自行实现接口。

  • inc 头文件目录

  • lib .lib 文件目录

  • pulses 也是头文件目录

移植外部模块库到ET12¶

  • 把库文件夹放到ET12工程目录,如下把driver目录 修改为 把库文件夹放到et12

  • 在工程中把.lib文件和et12目录下的驱动文件添加到工程

  • Option->Include Paths中添加 /pulses 和 inc/目录

  • 实现驱动文件的接口
