1.2.4 : Stream[4] 到 Steam[25]¶

16 Channels on 11 bits (0..2047)，16个通道，每个通道占用11位，值得范围为0~2047

16 Channels * 11Bits = 22 bytes

通道直

百分比

0

-125%

204

-100%

1024

+0%%

1843

+100%

2047

+125%

Values are concatenated to fit in 22 bytes like in SBUS protocol. Failsafe values have exactly the same range/values than normal channels except the extremes where 0=no pulse, 2047=hold. If failsafe is not set or RX then failsafe packets should not be sent.
